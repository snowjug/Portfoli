## Customization Guide

### Personal Information

1. **Name & Title**: Edit `index.html`
   - Line 27: Replace "Your Name" with your actual name
   - Line 28: Update "Creative Developer & Designer" with your title
   - Line 29: Update the description
   - Line 176: Update footer name

2. **Contact Links**: Edit `index.html`
   - Line 125: Replace LinkedIn URL with your profile
   - Line 135: Replace GitHub URL with your username
   - Line 145: Replace email with your email address
   - Line 155: Replace Google Drive link with your resume link

### Resume Link Setup

To get your Google Drive resume link:
1. Upload your resume PDF to Google Drive
2. Right-click the file â†’ Share â†’ Get link
3. Change permissions to "Anyone with the link can view"
4. Copy the link and replace it in line 155 of `index.html`

### Projects

Edit the project cards in `index.html` (lines 96-122):
- Update project titles
- Update project descriptions
- Add project links if needed

### Skills

Edit the skills section in `index.html` (lines 80-92):
- Update skill categories
- Update technologies

### Colors & Styling

Edit `styles.css` variables (lines 9-18):
```css
--primary-color: #000000;
--accent-color: #0071e3; /* Change to your preferred color */
```

## How to Use

1. Open `index.html` in a web browser to view locally
2. Deploy to GitHub Pages, Netlify, or Vercel for free hosting
3. Customize all placeholder content with your information
